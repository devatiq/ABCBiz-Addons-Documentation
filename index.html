  • Server Requirements

    PHP server properties

    Ensuring that your server is configured correctly is essential for the optimal functioning of our plugin. It's important to verify that specific PHP server properties are set as described below. If you're not familiar with server configurations or encounter any difficulties, we recommend reaching out to your hosting provider for assistance. They are well-equipped to ensure these settings are correctly implemented for the smooth operation of the plugin. Here's what you need to check:

    • PHP Version 7.4+
    • max_execution_time 120
    • memory_limit 128M
    • post_max_size 32M
    • upload_max_filesize 32M
    • max_input_vars 2000

  • Install ABCBiz Addons

    We have two ways to installing the plugin:

    Installation via WordPress

    • Step 1: Log in to your WordPress Dashboard.
    • Step 2: Navigate to 'Plugins' and click on 'Add New'.
    • Step 3: Above the plugin list, find and click the 'Upload Plugin' button.
    • Step 4: Choose the downloaded .zip file of ABCBiz Addons by clicking on 'Choose File'.
    • Step 5: Once the file is selected, click 'Install Now' to upload the plugin to your site.
    • Step 6: After the upload is complete, click 'Activate Plugin' to enable ABCBiz Addons in WordPress.

    Installation via FTP

    1. Step 1: Download the ABCBiz Addons plugin.

      2. Acquire the .zip file of the ABCBiz Addons plugin

    2. Step 2: Unzip the ABCBiz Addons plugin file.

      3. Locate the downloaded .zip file on your computer and unzip it. You should now have a folder named 'abcbiz-multi-addons-pro-for-elementor'.

    3. Step 3: Download and install an FTP client.

      4. Choose an FTP client like FileZilla, Cyberduck, or WinSCP, and install it on your computer.

    4. Step 4: Gather your FTP credentials.

      5. Obtain your FTP host, username, and password, usually provided by your web hosting service.

    5. Step 5: Connect to your server using the FTP client.

      6. Open your FTP client, enter your FTP details, and connect to your server.

    6. Step 6: Navigate to the WordPress plugins directory.

      7. In your FTP client, navigate to the 'wp-content/plugins' directory of your WordPress installation.

    7. Step 7: Upload the ABCBiz Addons plugin folder.

      8. Locate the unzipped 'abcbiz-multi-addons-pro-for-elementor' folder on your computer and upload it to the 'wp-content/plugins' directory on your server.

    8. Step 8: Verify the upload.

      9. Ensure that the 'abcbiz-multi-addons-pro-for-elementor' folder and its contents are correctly uploaded to the 'plugins' directory.

    9. Step 9: Activate the plugin in WordPress.

      10. Log into your WordPress dashboard, go to 'Plugins', find 'ABCBiz Addons for Elementor', and click 'Activate'.

    10. Step 10: Disconnect from the FTP server.

      11. After completing the upload and activation, safely disconnect from the FTP server.

  • WooCommerce Widgets

    To utilize WooCommerce widgets in ABCBiz Addons, you must first install the WooCommerce plugin on your WordPress site. Follow these steps to install WooCommerce:

    1. Step 1: Log in to your WordPress Dashboard.
    2. Step 2: Navigate to 'Plugins' and click on 'Add New'.
    3. Step 3: In the search bar, type 'WooCommerce' and press Enter.
    4. Step 4: Find the WooCommerce plugin in the search results and click 'Install Now'.
    5. Step 5: After the installation is complete, click 'Activate' to enable WooCommerce on your site.
    6. Step 6: Follow the WooCommerce setup wizard to configure your store.

    Once WooCommerce is installed and configured, you'll be able to use the WooCommerce widgets in ABCBiz Addons to enhance your online store.

  • One-click Importer

    To help you set up website with the least efforts, we include in the theme package dummy content file. After installing it, all content seen on the live demo will be imported.

    there are two ways to upload demo content, 1 Click Demo Install, and Importing manually .xml files. Both methods are described below:

    1. 1. Go to Appearance > Theme Options on the left menu.

    2. 2. Click on Import Demo Data. Note: Before you begin, make sure all the required plugins are activated.

    3. 3. Please wait, it can take a couple of minutes.

    4. 4. Import Slider Revolution:

      In order to import Revolution Slider, go to Revolution Slider > Import Slider and then select sliders demo files in folder /Sample_Data

    Now you have a website just like our demo.

  • Manual Importing

    To imprort the demo content by yourself, please follow the steps below:

    If the script auto importing demo data doesn't works with your host, we had included sample data within the theme files, take a look at the folder /Sample_Data

    1. 1. Go to Tools > Import. Use WordPress import plugin to import content.xml file
    2. 2. Import Revolution Slider Slide
    3. 3. Go to Theme Options > Import/Export to import theme_options.json files
    4. 4. Install Widget Settings Importer/Exporter plugin, find widgets.wie and then import widgets area.
    5. 5. Go to Appearance > Menus to set theme menu locations.

    Now you have a website just like our demo.
